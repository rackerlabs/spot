GCP Spot VMs
=============

Google Cloud does not offer an API to fetch latest Current Spot VMs Pricing, programmatically.

We can find the prices from the following page manually.

 - https://cloud.google.com/spot-vms/pricing#current_spot_vms_pricing

## Regions

| GCP Region                   | Rackspace Spot Region |
| ---------------------------- | --------------------- |
| lowa (us-central1)           | us-central-dfw-1      |
| Northern Virginia (us-east4) | us-east-iad-1         |
